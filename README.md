# my-prototypes
Add Array, Numbers, Object and String prototypes in your project.

**Install**
 ``` npm install my-prototypes ``` 

**Use**
 ```  require('my-prototypes').init()  ``` 

 Check the [documentation](https://github.com/marcosomma/my-prototypes/wiki) for the usage.
