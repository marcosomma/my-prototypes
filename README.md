# my-prototypes
Add String and Numbers prototypes in your project.

##### Install
```npm install my-prototypes```

##### Use
``` require('my-prototypes').init ```

### Strin prototypes
- htmlEscape
- htmlUnescape
- validateContent (dateEU, dateUS, time, email, ip-address, alphanumeric and phone)

### Number prototypes
- formatMoney
- getSecond
- getMilliSecond
